## Basic Typescript with Jest so Maria does not leave Gerry

### Pre Reqs / Setup
 * npm install typescript -g
 * npm install jest -g
 * npm install npx -g
 * npm install
 * npm install --save-dev

### Developing
 * npm run watch
  * alias for `tsc -w`

### Run .ts files
 * npx ts-node ./index.ts

### Testing
 * npm test
